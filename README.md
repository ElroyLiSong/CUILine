# UILine
Swift 实现的画线控件（支持xib）

因项目中经常用到各种分割线，索性封装了一个画线的控件。
## 特性
- 支持在xib中使用
- 支持设置线的颜色
- 支持设置线的粗细(单位：px)
- padding属性
- 支持横线和竖线
- 支持画虚线
- 可设置虚线间隔
- 可设置虚线线段的长度
